# RetentionOS

AI Content Strategist for Web3 Creators

## Framework

Vanilla JavaScript with Tailwind CSS (CDN)

## Preview

Configured to run on `0.0.0.0:8080` for k8s HTTPRoute access.

## Theme

Modern Dark theme with slate/zinc color palette, featuring:
- Deep dark backgrounds (#0a0a0f, #16161f)
- High contrast text for readability
- Indigo and purple gradient accents
- Soft borders and subtle shadows
- Clean, premium aesthetic optimized for Web3 creators

## Features

RetentionOS analyzes your tweets, threads, and content to provide:

1. **Hook Analysis** - Detect and improve your opening hook
2. **Retention Score** - 1-100 scoring based on engagement factors
3. **AI Detection** - Identify AI-generated phrases and patterns
4. **Human Rewrite** - Get natural, Web3-native rewrites
5. **Format Recommendation** - Optimal content format (tweet/thread/article)
6. **Audience Targeting** - Best audience segments for your content
7. **Missing Elements** - What to add for better performance
8. **Viral Potential** - Engagement forecast and prediction
9. **Posting Strategy** - When and how to post
10. **Improvement Suggestions** - 3 actionable tips to boost performance
